### Symbols

This chapter describes the symbolic names used in this document.

| **Element** | **Description** |
| --- | --- |
| [CC] | Uppercase ISO country code as assigned by the DNS standard for the cluster location, for further information please refer to |
| [TYPE] | Uppercase type of server as described in** ****4.2** |
| [YYY] | 3 letters of alphanumeric characters in uppercase from A to Z, usually abacus code |
| [XX] | 2 digit serial number from 00 and 99 |
| [XXX] | 3 digit serial number from 000 to 999 |
| [XXXX] | 4 digit serial number from 0000 to 9999 |
| [XXXXX] | 5 digit serial number from 00000 to 99999 |
| [XXXXXX] | 6 digits database serial number from 000000 to 999999 |
| [DDDDD] | Domain name, 3 to 5 alphanumeric characters in uppercase from A to Z |
| [DirNr] | 4 digits serial directory number from 0000 to 9999, these are the first 4 digits from the database serial number [XXXXXX] |
| [Home-Appl-Server] | Explicit abbreviated server name |
| [Home-HUB-Server] | Explicit abbreviated server name |
| [HomeServer] | Abbreviated Notes name of home/mail server and/or also administration server |
| [Username] | Abbreviated Notes User Name with Certificate and Organization, e.g. “Peter Mueller/User/Org” |
